# 道德经 六章

# 原文

谷神不死，是谓玄牝（pìn）。玄牝之门，是谓天地根。绵绵若存，用之不勤。

# 译文

神奇的空间作用是永远不会消失的，它好像是一个玄妙的母体。而这一母体的生殖器官，就是产生天地万物的根源。空间连绵不断似乎永远存在，万物使用它却又用不完它。

# 解读

老子认为空间包含着万物，就像孕育着子女的母体，因此，空间也是万物产生的基础，没有空间，也就没有万物。
